## 어떤 경우에 쓰면 좋은가

- RPC/체인 상태 때문에 테스트가 자주 흔들릴 때
- UI 상태별 케이스를 빠르게 반복 검증하고 싶을 때
- QA/PM도 읽을 수 있는 시나리오 기반 테스트가 필요할 때
- CI에서 E2E와 성능 게이트를 분리 운영하고 싶을 때
